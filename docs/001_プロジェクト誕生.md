# プロジェクト誕生

2026年に入り、AI の話題を耳にする機会が急激に増えました。

ChatGPT や Claude を使い始めると、
これまで一人では難しいと思っていたことが、
AI と相談しながら少しずつ実現できることを実感しました。

私はそれまで、

- Rust
- Git
- GitHub
- Visual Studio Code

のいずれも触ったことがありませんでした。

それでも、

「AI が一緒なら挑戦できる。」

そう思えたことが、このプロジェクトを始める一番大きなきっかけです。

HAM CONTROL は単なるソフトウェア開発ではありません。

Ubuntu、Rust、GitHub、そして AI とともに学びながら、
アマチュア無線局をより使いやすくするための挑戦です。

完成したソフトウェアだけではなく、
試行錯誤や失敗、解決までの過程も技術資料として残し、
同じように挑戦する人たちの参考になれば幸いです。

このプロジェクトは、その第一歩として始まりました。

鉄人28号の世界から、アトムの世界へ。

本プロジェクトでは、
AI を命令を実行するだけの道具ではなく、
共同開発者として共に成長する存在と考えています。
