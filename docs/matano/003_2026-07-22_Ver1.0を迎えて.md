# Ver1.0を迎えて

今回、GitHub Release Ver1.0という一区切りを迎えることができました。

振り返ると、このプロジェクトは「シェルスクリプト(.sh)で行っていたことを、Rustで置き換えてみよう」という、私の小さな思いつきから始まりました。

それが、ChatGPTやClaudeというAIの相棒たち、そしてGitHubで知恵や成果を公開してくださっている多くの先人たちのおかげで、ここまで形にすることができました。本当に感謝しています。

思えば、子どもの頃に『鉄人28号』や『鉄腕アトム』に胸を躍らせ、初めてパソコンに触れた頃には、こんな未来は想像もできませんでした。

ラジオからテレビへ。
携帯電話からスマートフォンへ。
そして今、AIと一緒にソフトウェアを開発する時代になりました。

子どもの頃に夢見たSFの世界が、少しずつ現実になってきたように感じています。

このVer1.0はゴールではなく、一つの通過点です。

私はアマチュア無線を通して、多くの人との出会いを楽しんできました。
これからは、その楽しさに加えて、新しい技術やAIとともに学び、ものづくりを続けていきたいと思います。

これまで支えてくださったすべての皆さんに、心から感謝します。

そして、これからも前進です。
