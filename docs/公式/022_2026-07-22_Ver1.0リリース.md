# 022 Ver1.0リリース (2026-07-22)

## 概要

ham_control_v02 は Ver1.0 として一区切りを迎えた。

本版では新機能の追加よりも品質向上を重視し、実運用を通して判明した課題を整理・改善した。

## Ver1.0の到達点

- WM_COPYDATA方式によるTurbo HAMLOG連携を実装
- WSJT-X / FreeDV / fldigi / MMSSTV の4ソースで実機動作確認
- ADIF出力品質向上
- CSV出力品質向上
- 重複QSO記録防止
- FreeDV再送対策
- ログ保存先を logs/ に整理
- GitHub Release Ver1.0 作成

## 今後

Ver1.0は安定版として保守を行う。

新機能については、必要性を十分に検討したうえで Ver1.1 以降で取り組む。
