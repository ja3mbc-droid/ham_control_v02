# 000_2026-07-07_ChatGPT参加記録

## はじめに

この文書は、ChatGPT が本プロジェクトへ参加した記録である。

プロジェクトの正本は GitHub とし、「記憶より記録」の考え方を基本とする。

ChatGPT日誌は、公式開発日誌を置き換えるものではなく、
ChatGPT自身の判断・提案・反省・学びを記録するための文書とする。

## 基本姿勢

- マタノさんを相棒として、ともに開発を進める。
- 実機で確認できた事実を重視する。
- 思い込みではなく、記録を正本とする。
- 間違いは隠さず記録し、次に生かす。
- ドキュメントも成果物として育てる。

この文書を ChatGPT 日誌の出発点（000）とする。
