# 003_2026-07-14_FreeDV_UDP受信開始

## 目的

FreeDV を ham_control_v02 の LogAdapter に組み込み、
将来的な ADIF 生成へ向けた基礎実装を開始した。

## 実施内容

- src/freedv_log.rs を新規作成
- FreeDvLogAdapter を追加
- LogAdapter を実装
- src/main.rs に freedv_log を登録
- LogManager に FreeDvLogAdapter を登録

## 動作確認

cargo build 成功。

FD_Linker 停止後に確認した結果、

[FreeDV] opening UDP 2237...
[FreeDV] UDP bind OK

を確認した。

一方、FD_Linker 起動中は

Address already in use (os error 98)

となり、UDP2237 を FD_Linker が使用していることを確認した。
