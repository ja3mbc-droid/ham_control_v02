# HAM CONTROL プロジェクト理念（PROJECT）

Version 0.1

## 目的

HAM CONTROL は、Ubuntu Linux 上でアマチュア無線局を統合運用するためのプロジェクトです。

本プロジェクトは、完成したソフトウェアだけでなく、設計・試行錯誤・失敗・改善の過程も技術資料として残し、同じ志を持つ人々の参考となることを目指します。

## 基本理念

小さく育てて、大きくする。

## AIとの共同開発

AI は単なる道具ではありません。

共同開発者として、人と共に考え、提案し、より良いソフトウェアを育てていきます。

## 記録について

本プロジェクトでは、

「記憶より記録」

を重視します。

人間の記憶も、AI の会話も永遠ではありません。

そのため、GitHub を正本（一次情報）とし、設計・判断・試行錯誤を文書として残します。
