# 実用版の位置づけ

## 目的

ham_control_v02 の開発では、「実用版」を正常動作の基準（リファレンス）とする。

実用版とは、実際の無線機・flrig・周辺ソフトと接続し、
運用できることを確認した版である。

## 開発版との関係

新しい機能は開発版で実装する。

実装後は、

実用版
↓
開発版

の順に同じ操作を行い、結果を比較する。

重要なのは、

「道が違っても結果は同じ」

である。

開発版は実用版と同じ結果になることを目標とする。

## 確認項目

・画面表示
・CAT制御
・PTT動作
・TIME_ON / TIME_OFF
・SWR
・Power
・各入力項目
・ログ出力
・HAMLOG / ADIF 連携

## 基本方針

推測では判断しない。

実機で確認し、
記録を残し、
Gitへ保存する。

実用版は、常に比較対象として維持する。
